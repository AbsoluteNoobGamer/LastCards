# Sound Files Reference

Place your `.wav` files in this folder with these **exact** names. The game will look for them as-is.

## Core gameplay

| Rename your file to | When it plays |
|--------------------|---------------|
| `Draw-Card.wav` | Player draws from deck (voluntary/penalty draw during play) |
| `deal_card_1.wav` | Dealer dealing to 1st player (first opponent, dealt first) |
| `deal_card_2.wav` | Dealer dealing to 2nd player |
| `deal_card_3.wav` | Dealer dealing to 3rd player |
| `deal_card_4.wav` | Dealer dealing to 4th player |
| `deal_card_5.wav` | Dealer dealing to 5th player |
| `deal_card_6.wav` | Dealer dealing to 6th player |
| `deal_card_7.wav` | Dealer dealing to 7th player |
| `deal_card_8.wav` | Dealer dealing to 8th player |
| `deal_card_9.wav` | Dealer dealing to 9th player |
| `deal_card_10.wav` | Dealer dealing to 10th player |
| … | **Deal order:** opponents first (clockwise), you last. For 2 players use 1–2; for 5 use 1–5; for 10 use 1–10. You always get the highest number. |
| `card_place.wav` | Any card played to discard pile |
| `card_select.wav` | Tapping a card in hand |
| `end_turn-button.wav` | Pressing the End Turn button |

## Penalties & turn

| Rename your file to | When it plays |
|--------------------|---------------|
| `penalty_draw.wav` | Penalty draw (2 / Black Jack) |
| `turn_start.wav` | Start of a player's turn |
| `timer_warning.wav` | 10 seconds left on turn timer |
| `timer_expired.wav` | Turn timer ran out |

## Win / loss

| Rename your file to | When it plays |
|--------------------|---------------|
| `player_win.wav` | Local player wins |
| `player_lose.wav` | Local player loses |
| `opponent_out.wav` | Opponent eliminated (tournament) |

## Tournament

| Rename your file to | When it plays |
|--------------------|---------------|
| `tournament_qualify.wav` | Player qualified for next round |
| `tournament_eliminate.wav` | Player eliminated from tournament |
| `tournament_win.wav` | Won the tournament |

## Bust mode

| Rename your file to | When it plays |
|--------------------|---------------|

## Other

| Rename your file to | When it plays |
|--------------------|---------------|

**Note:** Keep the exact spelling, hyphens, and underscores. Case matters on some platforms.
